The Tropical Bot

The tropical bot is your overall server bot. It has everything you need and want. It goes from kicking someone to creating your own commands! The Tropical bot is here for your server and you.

Cool Features

  • Enabling Features - Anti-link, Image Welcoming
  • Making your own welcome and goodbye messages
  • Making Costum Commands
  • Blacklisting words from your server

Welcome-Farewell Message Variables

  • {member} - Mentions the user
  • {server} - Shows current server name
  • {count} - Shows the current member count
